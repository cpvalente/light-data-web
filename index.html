light-data

    Minimal yet rich

    Light data is a web application for management of production kit lists.

    We aim on creating a tool for management and sharing of documentation in production worflows.

    • Keep track of equipment maintenance
    • Assign equipment to venue and share kit lists withs incoming guests
    • With powerful excel like features, use production kit lists to create production documentation
    • Allow external users to follow lists and make maintenance requests

    Made for entertainment

    • Theaters looking to manage and allocate equipment across venues.
    • Freelancers managing inventory and maintenance logging for service contracts.
    • Installation managers allowing logging and request of maintenance from technicians.

    Collaborative

    Light data is built on realtime databases, making it excellent for multiple users and collaborative setups.
