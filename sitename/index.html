light-data

    For entertainment

    Initially built with lighting and video systems in mind, light-data is meant to be used by

    Theaters looking to manage and allocate equipment across venues.

    Freelancers managing inventory and maintenance logging for service contracts.

    Installation managers allowing logging and request of maintenance from technicians.

    Minimal yet rich

    Light data is a web application for management of production kit lists.

    The aim is for a minimal and simple way to generate and share documentation.

    Collaborative

    Light data is built on realtime databases, making it excellent for multiple users and collaborative setups.

    Plays with others

    Integration with vectorworks.
